# Smart-Air-Quality-Monitoring-using-ESP32
The Smart Air Quality Monitor is an embedded IoT system designed to continuously measure harmful gas concentrations alongside temperature and humidity. Readings are displayed on  LCD screen and a buzzer provides an audible alarm when pollution exceeds dangerous thresholds.


## Overview
This project monitors air quality, temperature, and humidity using ESP32.

## Components
- ESP32
- MQ135 Gas Sensor
- DHT11
- LCD (I2C)
- Buzzer

## Working
MQ135 detects gas levels.
DHT11 measures temperature and humidity.
ESP32 displays data on LCD.
Buzzer turns ON when gas exceeds threshold.
